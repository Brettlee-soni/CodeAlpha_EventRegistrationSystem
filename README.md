## Project Overview

The Event Registration System is a backend application developed using Spring Boot and MySQL. It allows users to create and manage events, register participants, update event details, view registrations, and cancel registrations through RESTful APIs.

### Features
- Create Events
- View Events
- Update Events
- Delete Events
- Register Participants
- View Registrations
- Cancel Registrations
- MySQL Database Integration
- REST API Architecture

### Technologies Used
- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Postman
